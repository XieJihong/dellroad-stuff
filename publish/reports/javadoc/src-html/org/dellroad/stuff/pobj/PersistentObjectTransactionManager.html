028/**
029 * "Disruptor" style transaction manager for {@link PersistentObject} databases.
030 *
031 * <h3>Overview</h3>
032 *
033 * <p>
034 * Instances of this class maintain a dedicated thread for performing transactions on a specific {@link PersistentObject}
035 * database. The {@link #performTransaction performTransaction()} and {@link #scheduleTransaction scheduleTransaction()}
036 * methods schedule synchronous and asynchronous transactions (respectively) to be executed by this thread.
037 * </p>
038 *
039 * <p>
040 * By performing all transactions on the same thread, the {@link PersistentObject} root object is stays in the cache
041 * of a single CPU core, improving performance ("Disruptor" style). The single threading also guarantees transaction
042 * serialization, atomicity, and isolation. The capacity of the queue that holds upcoming transactions can be configured
043 * via {@link #setQueueCapacity setQueueCapacity()} (default {@link Integer#MAX_VALUE}). When the queue is full,
044 * scheduling a new transaction blocks until space is available.
045 * </p>
046 *
047 * <p>
048 * Within a transaction, {@link #getRoot} is used to access that transaction's {@link PersistentObject} root object.
049 * The returned object graph is private to that thread's transaction (except in shared mode; see below).
050 * </p>
051 *
052 * <h3>Transaction Types</h3>
053 *
054 * <p>
055 * Transactions may be read-only or read-write. Read-only transactions may optionally be configured to access a
056 * {@linkplain PersistentObject#getSharedRoot shared root}, in which case all read-only transactions share the same
057 * object graph (so no per-transation copies are required), but these transactions must also be careful not to modify
058 * the object graph returned by {@link #getRoot}. For read-write transactions, any modifications to the
059 * object graph returned by {@link #getRoot} will be persisted when the transaction completes (assuming it validates).
060 * In either case, references to the object graph returned by {@link #getRoot} may be returned to callers outside of the
061 * transaction, who may then safely access them with the assurance that they won't be modified by another thread (although in the
062 * {@linkplain PersistentObject#getSharedRoot shared root} case, modifications must still be avoided). Of course, once
063 * the transaction ends the transaction's object graph is no longer guaranteed to be up-to-date.
064 * </p>
065 *
066 * <p>
067 * Normally all changes to the associated {@link PersistentObject} database are being handled via this class: if not, and
068 * another thread updates the {@link PersistentObject} while a transaction is being handled,
069 * a {@link PersistentObjectVersionException} can result (the transaction may then be retried if desired).
070 * </p>
071 *
072 * <h3>Transaction Reentrancy</h3>
073 *
074 * <p>
075 * Transactions are reentrant, so that if {@link #performTransaction performTransaction()} is invoked by a thread while
076 * executing within an existing transaction, no new transaction is created; instead, execution proceeds in the context of
077 * the existing transaction. However, if the second transaction is not compatible with the first (e.g., a read-write
078 * transaction inside a read-only one), an exception is thrown.
079 * </p>
080 *
081 * <h3>Multiple Transactions</h3>
082 *
083 * <p>
084 * A thread executing within one {@link PersistentObjectTransactionManager}'s transaction may safely invoke
085 * {@link #performTransaction performTransaction()} on another {@link PersistentObjectTransactionManager}; however,
086 * the first transaction will block while the second executes, and callers must be careful to always nest these
087 * transactions in the same order to avoid possible deadlock.
088 * </p>
089 *
090 * <p>
091 * To allow distingushing among multiple {@link PersistentObjectTransactionManager} instances that may have open transactions
092 * at the same time, instances may be given unique {@linkplain #getName names} (names are automatically inherited from the
093 * {@linkplain #setBeanName bean name} when instances are declared in a Spring bean factory). Within an open transaction,
094 * the associated {@link PersistentObjectTransactionManager} can be accessed by name via {@link #getCurrent(String)}.
095 * As a convenience, in the common case of there only being one open transaction in the current thread, the
096 * {@link #getCurrent()} variant works; dealing with names is not necessary when there is only one instance in play.
097 * </p>
098 *
099 * <h3>Annotation Usage</h3>
100 *
101 * <p>
102 * The easiest way to use this class is to annotate transactional methods with the
103 * {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} annotation and apply the associated AOP aspect,
104 * and then use {@link #getCurrent()} within the transaction to access the {@link PersistentObjectTransactionManager} and
105 * the associated transaction root object. For example:
106 * <pre>
107 *  <b>&#64;PersistentObjectTransactional</b>
108 *  public User loginUser(String username) throws LoginException {
109 *      final MyDatabase database = <b>PersistentObjectTransactionManager.&lt;MyDatabase&gt;getCurrent().getRoot()</b>;
110 *      for (User user : database.getUsers()) {
111 *          if (user.getUsername().equals(username)) {
112 *              if (user.getLoginCount() &gt;= MAX_CONCURRENT_LOGINS)
113 *                  throw new LoginException("too many logins");
114 *              user.setLoginCount(user.getLoginCount() + 1);
115 *              return user;
116 *          }
117 *      }
118 *      throw new LoginException("user not found");
119 *  }
120 * </pre>
121 * Don't forget to configure the {@link PersistentObjectTransactional &#64;PersistentObjectTransactional} AOP aspect
122 * via Spring as described in {@link PersistentObjectTransactional &#64;PersistentObjectTransactional}.
123 * </p>
124 *
125 * @param <T> root object type
126 * @see PersistentObjectTransactional &#64;PersistentObjectTransactional
127 */
128@Component
129public class PersistentObjectTransactionManager<T> extends AbstractBean implements BeanNameAware {

169    @Override
170    public void afterPropertiesSet() throws Exception {
171        super.afterPropertiesSet();
172
173        // Check name
174        if (this.name == null)
175            throw new IllegalStateException("no bean name configured");
176
177        // Initialize queue
178        this.queue = new LinkedBlockingQueue<FutureTask<?>>(this.getQueueCapacity());
179
180        // Start worker thread
181        if (this.workerThread != null)
182            throw new IllegalStateException("instance is already started");
183        this.workerThread = new WorkerThread();
184        this.workerThread.start();
185    }
186
187    @Override
188    public void destroy() throws Exception {
189
190        // Stop worker thread
191        if (this.workerThread == null)
192            throw new IllegalStateException("instance is already shutdown");
193        this.queue.add(TERMINATE_TASK);
194
195        // Wait for worker thread to finish
196        this.workerThread.join();
197        this.workerThread = null;
198
199        // Reset queue
200        this.queue = null;
201
202        // Done
203        super.destroy();
204    }

272    /**
273     * Schedule a transaction, wait for it to complete, and return the result.
274     *
275     * <p>
276     * If the current thread is already executing within a transaction associated with this instance, then
277     * no new transaction is created; instead, the {@code action} executes in the context of the existing
278     * transaction.
279     * </p>
280     *
281     * @param action action to perform within the transaction
282     * @param readOnly true for a read-only transaction
283     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
284     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
285     * @return result of executing the {@code action}
286     * @throws IllegalArgumentException if {@code action} is null
287     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
288     *  or has been {@linkplain #destroy} shut down
289     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
290     *  and {@code readOnly} is false
291     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
292     *  and {@code shared} is false
293     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate
294     * @throws PersistentObjectVersionException if {@code readOnly} is false and the database was modified by another thread
295     * @throws PersistentObjectException if {@code readOnly} is false and some other error occurs
296     * @throws Exception if thrown by {@code action}
297     */
298    public <V> V performTransaction(Callable<V> action, boolean readOnly, boolean shared) throws Exception {
299        if (this.isReentrant(readOnly, shared))
300            return action.call();
301        return this.performTransaction(this.scheduleTransaction(action, readOnly, shared));
302    }

304    /**
305     * Schedule a transaction and wait for it to complete.
306     *
307     * @param action action to perform within the transaction
308     * @param readOnly true for a read-only transaction
309     * @param shared if {@code readOnly} is true, access the {@linkplain PersistentObject#getSharedRoot shared root}
310     *  to avoid a copy operation; in this case, {@code action} must not make any modifications
311     * @throws IllegalArgumentException if {@code action} is null
312     * @throws IllegalStateException if this instance is not yet {@linkplain #afterPropertiesSet started}
313     *  or has been {@linkplain #destroy} shut down
314     * @throws IllegalStateException if this instance already has a read-only transaction open in the current thread
315     *  and {@code readOnly} is false
316     * @throws IllegalStateException if this instance already has a shared mode read-only transaction open in the current thread
317     *  and {@code shared} is false
318     * @throws PersistentObjectValidationException if {@code readOnly} is false and the modified root does not validate
319     * @throws PersistentObjectVersionException if {@code readOnly} is false and the database was modified by another thread
320     * @throws PersistentObjectException if {@code readOnly} is false and some other error occurs
321     */
322    public void performTransaction(Runnable action, boolean readOnly, boolean shared) {
323        if (this.isReentrant(readOnly, shared)) {
324            action.run();
325            return;
326        }
327        this.performTransaction(this.scheduleTransaction(action, readOnly, shared));
328    }

330    /**
331     * Get the instance associated with an open transaction in the current thread and having the given manager name.
332     *
333     * @param name name assigned to the desired instance
334     * @throws IllegalArgumentException if {@code name} is null
335     * @throws IllegalStateException if the current thread is not running within a transaction managed by
336     *  a {@link PersistentObjectTransactionManager} assigned the given name
337     */
338    @SuppressWarnings("unchecked")
339    public static <T> PersistentObjectTransactionManager<T> getCurrent(String name) {
340        if (name == null)
341            throw new IllegalArgumentException("null name");
342        final HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
343        final PersistentObjectTransactionManager<?> manager = managerMap != null ? managerMap.get(name) : null;
344        if (manager == null) {
345            throw new IllegalStateException("no PersistentObjectTransactionManager named `" + name
346              + "' has an open transaction in the current thread");
347        }
348        return (PersistentObjectTransactionManager<T>)manager;
349    }

351    /**
352     * Get the (unique) instance associated with an open transaction in the current thread.
353     *
354     * <p>
355     * This is a convenience method for the common case where there is only one instance associated with the current thread.
356     * </p>
357     *
358     * @throws IllegalStateException if the current thread is not running within a
359     *  {@link PersistentObjectTransactionManager} transaction
360     * @throws IllegalStateException there is more than one {@link PersistentObjectTransactionManager} transaction
361     *  open in the current thread
362     */
363    @SuppressWarnings("unchecked")
364    public static <T> PersistentObjectTransactionManager<T> getCurrent() {
365        final HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
366        if (managerMap == null || managerMap.isEmpty()) {
367            throw new IllegalStateException("there are no PersistentObjectTransactionManager transactions open"
368              + " in the current thread");
369        }
370        if (managerMap.size() > 1) {
371            throw new IllegalStateException("there is more than one PersistentObjectTransactionManager transaction open"
372              + " in the current thread");
373        }
374        return (PersistentObjectTransactionManager<T>)managerMap.values().iterator().next();
375    }

419// Internal methods

421    private boolean isReentrant(boolean readOnly, boolean shared) {
422
423        // Are we already within a transaction?
424        final TxInfo txInfo = this.currentRoot.get();
425        if (txInfo == null)
426            return false;
427
428        // Check for incompatibilities
429        if (txInfo.isReadOnly()) {
430            if (!readOnly) {
431                throw new IllegalStateException("illegal read-write transaction created reentrantly"
432                  + " within a read-only transaction");
433            }
434            if (txInfo.isShared() && !shared) {
435                throw new IllegalStateException("illegal non-shared read-only transaction created reentrantly"
436                  + " within a shared read-only transaction");
437            }
438        }
439        return true;
440    }

442    private <V> FutureTask<V> scheduleTransaction(FutureTask<V> task) {
443
444        // Sanity check
445        if (task == null)
446            throw new IllegalArgumentException("null task");
447
448        // Enqueue transaction
449        try {
450            this.queue.add(task);
451        } catch (NullPointerException e) {
452            throw new IllegalStateException("instance not started or has been shutdown");
453        }
454
455        // Done
456        return task;
457    }
458
459    private <V> V performTransaction(Future<V> future) {
460        boolean interrupted = false;
461        try {
462            while (true) {
463                try {
464                    return future.get();
465                } catch (InterruptedException e) {
466                    interrupted = true;
467                }
468            }
469        } catch (ExecutionException e) {
470            throw PersistentObjectTransactionManager.<RuntimeException>maskException(e.getCause()); // re-throw original exception
471        } catch (CancellationException e) {
472            throw new RuntimeException("transaction method execution was canceled", e);             // should never happen
473        } finally {
474            if (interrupted)
475                Thread.currentThread().interrupt();
476        }
477    }
478
479    private <V> Callable<V> getTask(final Callable<V> action, final boolean readOnly, final boolean shared) {
480        return new Callable<V>() {
481            @Override
482            public V call() throws Exception {
483                final PersistentObject<T>.Snapshot snapshot = readOnly && shared ?
484                  PersistentObjectTransactionManager.this.persistentObject.getSharedRootSnapshot() :
485                  PersistentObjectTransactionManager.this.persistentObject.getRootSnapshot();
486                final TxInfo txInfo = new TxInfo(snapshot, readOnly, shared);
487                return PersistentObjectTransactionManager.this.currentRoot.invoke(txInfo, new Callable<V>() {
488                    @Override
489                    public V call() throws Exception {
490                        return PersistentObjectTransactionManager.this.executeTransaction(action, txInfo);
491                    }
492                });
493            }
494        };
495    }
496
497    private Callable<Void> getTask(final Runnable action, boolean readOnly, boolean shared) {
498        return this.getTask(new Callable<Void>() {
499            @Override
500            public Void call() {
501                action.run();
502                return null;
503            }
504        }, readOnly, shared);
505    }
506
507    private <V> V executeTransaction(Callable<V> action, TxInfo txInfo) throws Exception {
508
509        // Associate this instance with the current thread
510        HashMap<String, PersistentObjectTransactionManager<?>> managerMap = ASPECT_MANAGER_MAP.get();
511        final boolean emptyMap = managerMap == null;
512        if (emptyMap) {
513            managerMap = new HashMap<String, PersistentObjectTransactionManager<?>>();
514            ASPECT_MANAGER_MAP.set(managerMap);
515        }
516        if (managerMap.containsKey(this.name)) {
517            throw new IllegalStateException("A PersistentObjectTransactionManager named `" + this.name
518              + "' already has an open transaction in the current thread; all names must be distinct");
519        }
520        managerMap.put(this.name, this);
521
522        // Perform transaction
523        V result;
524        try {
525
526            // Invoke callback
527            try {
528                result = action.call();
529            } catch (Exception e) {
530                throw e;
531            }
532
533            // Update database (read-write transactions only)
534            if (!txInfo.isReadOnly())
535                this.persistentObject.setRoot(txInfo.getSnapshot().getRoot(), txInfo.getSnapshot().getVersion());
536        } finally {
537            if (emptyMap)
538                ASPECT_MANAGER_MAP.remove();
539            else
540                managerMap.remove(this.name);
541        }
542
543        // Done
544        return result;
545    }

550        throw (T)t;

553// Worker thread
554
555    private class WorkerThread extends Thread {
556
557        private final Logger log = LoggerFactory.getLogger(this.getClass());
558
559        public WorkerThread() {
560            super("PersistentObjectTransactionManager-" + PersistentObjectTransactionManager.THREAD_COUNTER.incrementAndGet());
561        }
562
563        @Override
564        public void run() {
565            this.log.debug(this + " transaction thread starting");
566            int count = 0;
567            try {
568                while (true) {
569
570                    // Get next in queue
571                    final FutureTask<?> task = PersistentObjectTransactionManager.this.queue.take();
572                    if (task == TERMINATE_TASK)
573                        break;
574
575                    // Execute task
576                    task.run();
577                    count++;
578                }
579            } catch (Throwable t) {
580                this.log.error(this + " transaction thread caught unexpected exception", t);
581            } finally {
582                this.log.debug(this + " transaction thread stopping after " + count + " transactions");
583            }
584        }
585    }
